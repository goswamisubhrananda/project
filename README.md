# Airline Price Prediction Analysis Dashboard

## Project Overview
This project analyzes airline ticket prices using data visualization tools. The main goal is to understand how different factors such as airline, booking time, departure time, source and destination, and travel class affect ticket prices.

Interactive dashboards were created using **Power BI** and **Tableau** to visualize these patterns and provide insights useful for marketing managers and decision-makers in the aviation industry.

---

## Objectives
- Analyze airline ticket prices using real dataset.
- Understand how ticket prices vary across different airlines.
- Examine how booking time affects ticket pricing.
- Study the relationship between departure/arrival time and ticket price.
- Compare ticket prices between Economy and Business class.
- Build interactive dashboards using Power BI and Tableau.

---

## Dataset Description
The dataset contains information about airline ticket prices along with flight details.

### Dataset Fields

| Column Name | Description |
|-------------|-------------|
| airline | Name of the airline |
| source_city | City from where the flight departs |
| destination_city | City where the flight arrives |
| departure_time | Scheduled departure time |
| arrival_time | Scheduled arrival time |
| stops | Number of stops during the journey |
| class | Travel class (Economy / Business) |
| duration | Total travel duration |
| days_left | Number of days left before departure when booking was made |
| price | Ticket price |

## Dashboard Analysis

### 1. Price Variation by Airline
The dashboard shows how average ticket prices differ across airlines.

### 2. Booking Time vs Ticket Price
Ticket prices increase when tickets are purchased closer to the departure date.

### 3. Departure Time Impact
Flights scheduled during peak hours often have higher prices.

### 4. Source and Destination Analysis
Different routes show different pricing patterns depending on demand and distance.

### 5. Economy vs Business Class
Business class tickets are significantly more expensive than economy class due to additional services and comfort.

---

## Key Insights
- Airline companies follow different pricing strategies.
- Last-minute bookings tend to be more expensive.
- Flight timing influences ticket prices.
- Certain routes consistently have higher ticket fares.
- Business class fares are much higher than economy class.
